# Encarpetado-de-React.github.io

Proyecto estático organizado para GitHub Pages.

## Estructura

- `index.html`: entrada principal con el CRUD.
- `pages/`: páginas secundarias del laboratorio.
- `assets/css/`: estilos compartidos.
- `assets/js/`: lógica JavaScript de cada módulo.
- `logosena.png`: imagen usada por todas las páginas.

## Páginas

- `index.html`
- `pages/callback.html`
- `pages/promesas.html`
- `pages/async-await.html`
- `pages/object-destructuring.html`
